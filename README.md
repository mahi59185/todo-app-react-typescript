## 🚀 Live Demo

[View Live Application](https://todo-app-react-typescript-ashy.vercel.app/)

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- CSS

## ✨ Features

- Delete tasks
- Mark tasks as completed
- Track todos using custom hooks
- Responsive user interface
